## 🚀 Features

- ✅ **10 Multiple-Choice Questions** – Test your knowledge on HTML, CSS, and JavaScript
- ✅ **Shuffle Questions** – Questions appear in random order every time
- ✅ **Shuffle Options** – Answer options are randomized
- ✅ **30-Second Timer** – Each question has a time limit
- ✅ **Timer Warning** – Timer turns red and blinks when 5 seconds remain
- ✅ **Auto-Fail on Timeout** – If time runs out, question is marked wrong
- ✅ **Progress Bar** – Visual indicator of quiz completion
- ✅ **Instant Feedback** – Correct answers turn green, wrong answers turn red
- ✅ **Score Tracking** – Real-time score updates
- ✅ **Results Summary** – Shows final score, percentage, and performance message
- ✅ **Performance Message** – Personalized message based on score
- ✅ **High Score Tracking** – Best score saved in browser using localStorage
- ✅ **Review Answers** – See all your answers with correct answers
- ✅ **Restart Quiz** – Reset and play again
- ✅ **Responsive Design** – Works on desktop, tablet, and mobile devices

## 🛠️ How to Run Locally

### Option 1: Using Live Server (Recommended)
1. Open the project folder in VS Code
2. Right-click on `index.html`
3. Select **"Open with Live Server"**
4. The quiz will open in your browser

### Option 2: Direct Open
1. Navigate to the project folder
2. Double-click `index.html`
3. The quiz will open in your default browser

### Option 3: Clone from GitHub
1. git clone https://github.com/darkfear2005or2006-ai/quiz-app-project.git
2. cd quiz-app-project
3. open index.html


### How to Use
Open the quiz link
Read the question carefully
Select one of the 4 options (A, B, C, or D)
You will get immediate feedback:
✅ Green = Correct answer
❌ Red = Wrong answer (correct answer will be shown)
Click "Next Question" to proceed
You have 30 seconds per question (timer turns red at 5 seconds)
At the end, see your:
Final score
Percentage
Performance message
High score
All answers with correct answers
